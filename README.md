# Sistema de Gestión de Gimnasio

Bienvenido a la documentación oficial del sistema. Este proyecto es una solución integral diseñada para la administración eficiente de gimnasios, centralizando el registro de miembros, la gestión de planes y el control de pagos.

## Características Principales
El sistema cubre los flujos operativos esenciales:
- Gestión de Miembros: Registro de nuevos clientes, actualización de información y bajas.
- Planes de Entrenamiento: Configuración de tipos de membresías, costos y duración.
- Control de Caja: Procesamiento de pagos y verificación del estado de solvencia de los miembros.
- Asistencias: Control de asistencia de los socios.
- Configuración: Ajustes básicos del sistema.

---

## Requisitos Previos
Para ejecutar este sistema, asegúrate de tener instalado:
- Node.js v18 o superior
- npm o yarn
- PostgreSQL (o MySQL, según configuración)
- Git
